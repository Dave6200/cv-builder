# CV Builder — Project Context

## Что это за проект
CV Builder — веб-приложение для создания резюме. React + localStorage + PDF export.

## Стек
- React (Create React App)
- React Router DOM
- html2canvas + jsPDF для PDF экспорта
- Чистый CSS (без Tailwind)

## Структура файлов
- src/App.js — главный файл, роутинг, Builder компонент
- src/Landing.js — лендинг страница
- src/Splash.js — анимация загрузки
- src/TemplateModern.js — современный шаблон резюме
- src/TemplateMinimal.js — минималистичный шаблон
- src/TemplateClassic.js — классический шаблон (две колонки)
- src/App.css — все стили

## Дизайн система
- Основной цвет: #6c63ff (фиолетовый)
- Фон: #0f0f13 (тёмный)
- Акцент: #9c63ff
- Шрифт: Inter (форма), Playfair Display (заголовки)

## Правила
- Всегда давай полный код файла, никогда не давай частичный
- Дизайн тёмный, современный, минималистичный
- Не ломай существующую структуру компонентов
- При изменении шаблонов сохраняй проп data
- CSS анимации предпочтительнее JS анимаций
